Bootstrap Admin Dashboard

A responsive and modern admin dashboard developed using Bootstrap 5, HTML, CSS, and JavaScript. This project demonstrates dashboard UI design principles with a clean layout and user-friendly interface.

Features:
✔ Responsive Sidebar Navigation
✔ Revenue Analytics Section
✔ Customer Statistics Cards
✔ Performance Tracking Panel
✔ Claims Ratio Chart
✔ Bootstrap Components Integration
✔ Mobile-Friendly Layout
✔ Clean and Modern UI Design

Technologies Used:
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

Purpose:
This project was created to practice frontend development skills, Bootstrap components, responsive design, dashboard layouts, and UI development best practices.
